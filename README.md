# 2026电赛备赛规划
## 比赛简介
电赛全名全国大学生电子设计竞赛，于每年的一月份官网发布通知，五月底结束报名，报名时三人组队，七月份底公布器件清单，随后不久官网公布赛题，根据题目要求利用进行四天三夜设计一个作品并撰写设计报告，再将作品封箱在各赛区指定地点进行评测，最后还要以队单位参加综合测评，时间从早上八点到下午三点，单数年一次竞赛分评省奖和国奖，双数年为邀请赛一次评奖，均在全国参赛学校中评选。

## 控制类备赛

### 平台-小车

小车搭建所需模块：

- 系统板：
  - STM32: 赛题不一定允许使用，2020、2022年限制使用MSP
  - MSP430/432: 资料少，部分场景性能低，贵
- 电源模块：
  - DCDC开关电源模块
  - LDO线性稳压器
  - 接线：12v电源-DCDC-5v传感器-LDO-3.3v单片机和传感器
- 直流电机驱动模块：
  - 需要注意参数：输入电压、逻辑电压、驱动电流、最大功率
- 直流减速电机、编码器模块：
  - 一定要购买配有编码器的电机
- 舵机模块：
  - 若对控制精度和速度有要求，可以选择FOC无刷电机
- 传感器：
  - 灰度传感器：循线用
  - 陀螺仪模块
- 摄像头：
  - MT9V034: C语言编程，上限高，便宜
  - openMV: 使用micropython，编程简单，性能差，价格贵
- 通信模块：
  - HC-05蓝牙模块
