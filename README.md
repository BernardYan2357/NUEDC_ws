# 2026电赛备赛规划
## 比赛简介
电赛全名全国大学生电子设计竞赛，于每年的一月份官网发布通知，五月底结束报名，报名时三人组队，七月份底公布器件清单，随后不久官网公布赛题，根据题目要求利用进行四天三夜设计一个作品并撰写设计报告，再将作品封箱在各赛区指定地点进行评测，最后还要以队单位参加综合测评，时间从早上八点到下午三点。
