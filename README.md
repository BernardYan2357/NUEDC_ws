# 2026电赛备赛规划

## 比赛简介
<img width="208" height="208" alt="image" src="https://github.com/user-attachments/assets/1b19d54a-e45b-491d-8597-cd7edbd7c763" />


电赛全名全国大学生电子设计竞赛，单数年一次竞赛分评省奖和国奖，双数年为邀请赛一次评奖，均在全国参赛学校中评选

## 时间表

- 一月份官网发布通知
- 五月底结束报名
- 七月份底公布器件清单
- 随后不久官网公布赛题
- 根据题目要求利用进行四天三夜设计一个作品并撰写设计报告
- 从早上八点到下午三点将作品封箱在各赛区指定地点进行评测，最后还要以队单位参加综合测评

## 赛题分类与分析
一共有仪器类、通信类	、测控类、电源类、信号处理、控制类、飞行器、高频及通信类这几种赛题，考虑到专业相关、个人兴趣以及上手难度，我们将控制类赛题作为首选，但是与此同时在可能没有控制题的情况下，我们在时间富余情况下，还会进行电源题以及测控题的练习

### 时间规划
我们备赛时间大概八个月，主要分为三个阶段
#### 第一阶段：基础学习
- 机械与硬件部分
  -  熟悉使用solidworks去设计赛题所需实践平台
  -  熟练运用嘉立创EDA以及AD完成电路设计，在学习过程中了解元器件选型以及布局
- 电机控制
  - 了解比赛中常用的电机
  - 掌握PID、MPC、LQR等控制算法
  - 熟悉一些常见的控制平台如STM32、TI控制芯片等等
- 视觉部分
  - 了解常见Linux系统开发板如树莓派、Jestson nano、地瓜派等
  - 学习ROS2、OpenCV、深度学习以及Yolo相关知识 
#### 第二阶段：模块专项训练
两到三个星期为一个周期，完成一个综合性项目，同时在项目中完成流程的固化“选题 -> 方案设计 -> 分工 -> 开发 -> 联调 -> 总结”
#### 第三阶段：比赛模拟以及冲刺优化
进行至少2次为期4天3夜的真题模拟。严格模拟比赛环境，独立完成从选题到报告提交的全过程。
每次模拟后进行深度复盘，重点优化方案选择策略、时间分配和团队协作模式。
整理常用电路、代码模板、文档框架，形成团队的“知识手册”。

## 控制类备赛

### 平台-小车

小车搭建所需模块：
- 底盘：
  - 三轮：具有更好的转向能力，适用于直角转弯，21年题
  - 四轮：速度更快，24年题
- 系统板：
  - STM32: 赛题不一定允许使用，2020、2022年限制使用MSP
  - MSP430/432: 资料少，部分场景性能低，价格贵
- 电源模块：
  - DCDC开关电源模块
  - LDO线性稳压器
  - 接线：12v电源-DCDC-5v传感器-LDO-3.3v单片机和传感器
- 直流电机驱动模块：
  - 需要注意参数：输入电压、逻辑电压、驱动电流、最大功率
- 直流减速电机、编码器模块：
  - 一定要购买配有编码器的电机
- 舵机模块：
  - 若对控制精度和速度有要求，可以选择FOC无刷电机
- 传感器：
  - 灰度传感器：循线用，无MCU
  - 陀螺仪模块
- 摄像头：
  - MT9V034: C语言编程，上限高，便宜
  - openMV: 使用micropython，编程简单，性能差，价格贵
- 通信模块：
  - HC-05蓝牙模块
